## Media Query

#### `CSS Media Query`는 개발자가 지정한 규칙에 브라우저 및 장치 환경이<br>일치하는 경우에만 CSS가 적용할 수 있는 방법을 제공한다. 반응형 웹 디자인의 가장 핵심적인 부분!

### 미디어 쿼리 기본
+ 가장 간단한 미디어 쿼리 구무은 다음과 같다.

```css
@media media-type and (media-feature-rule){ /* 여기 부분은 조건 부분! */
    /* CSS rules go here! */
}
```
#

### 미디어 쿼리 장치 너비 비교

+ 320 ~ 480px 모바일
+ 481 ~ 768px 태블릿
+ 769 ~ 1024px 노트북
+ 1025 ~ 1200px 데스크탑
+ 1201px(이상) 데스크탑 및 TV
